## Getting Started

### Environment variables

To run the application, you need to set all the **[environment variables](.env)**

* Cloning

```commandline
git clone https://github.com/Umbreella/RLT_test_task.git
```

* Install requirements

```commandline
poetry install
```

* Run bot

```commandline
python main.py
```
